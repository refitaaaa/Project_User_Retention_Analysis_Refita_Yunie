# Project_User_Retention_Analysis_Refita_Yunie
Notebook ini berisi analisis retensi pengguna untuk memahami perilaku dan loyalitas pelanggan dari waktu ke waktu. Meliputi pembersihan data, analisis cohort, serta visualisasi tren retensi menggunakan Python (Pandas, NumPy, Matplotlib, Seaborn) di Google Colab untuk menghasilkan wawasan strategis berbasis data.
