# Project_User_Retention_Analysis_Refita_Yunie
Proyek ini menganalisis retensi pengguna untuk memahami perilaku dan loyalitas pelanggan dari waktu ke waktu. Melibatkan pembersihan data, analisis cohort, dan visualisasi tren retensi menggunakan Python (Pandas, NumPy, Matplotlib, Seaborn) di Google Colab, menghasilkan insight strategis berbasis data yang siap mendukung pengambilan keputusan bisnis.
